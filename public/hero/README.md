# Hero Carousel Images

Add your hero slide images to this folder:

- `slide-1.jpg` – First slide
- `slide-2.jpg` – Second slide  
- `slide-3.jpg` – Third slide

Supported formats: JPG, PNG, WebP. Recommended size: 1920×600px or similar aspect ratio.

To add or change slides, edit the `heroSlides` array in `src/pages/Index.tsx`.
